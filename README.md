# Bellabeat Smart Device Analysis (R & Tableau)

## Project Overview

This project analyzes Fitbit smart device usage data to generate actionable marketing and product insights for Bellabeat, a women-focused health technology company.

The objective was to identify behavioral patterns in user activity and translate them into strategic business recommendations.

---

## Business Context

Bellabeat is interested in understanding how users engage with smart devices in order to improve marketing strategy and product positioning.

This analysis focuses on identifying:

- Activity trends
- Engagement consistency
- Usage segmentation
- Opportunities for targeted marketing

---

## Tools & Technologies

- R (tidyverse, lubridate, janitor, skimr)
- Data cleaning and exploratory data analysis
- Tableau (for visualization and dashboarding)

---

## Dataset

- 33 Fitbit users
- Daily activity data
- Sleep tracking data
- Steps, calories, intensity levels

Full dataset details available in `dataset_description.md`.

---

## Analytical Approach

1. Cleaned and standardized column names
2. Converted date fields to proper formats
3. Removed duplicates and handled missing values
4. Aggregated daily activity metrics
5. Performed exploratory analysis
6. Identified engagement and behavior patterns
7. Developed business-focused recommendations

---

## Key Insights

- Users with consistent daily engagement showed higher overall activity levels.
- Variability in activity patterns suggests segmentation opportunities.
- Sleep and activity trends can inform personalized marketing messaging.
- Moderate users represent the strongest growth opportunity segment.

---

## Business Recommendations

- Emphasize habit-building features in marketing campaigns.
- Segment users based on engagement frequency.
- Target moderately active users with personalized growth messaging.
- Highlight lifestyle consistency rather than extreme performance metrics.
